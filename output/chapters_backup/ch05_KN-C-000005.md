Part 6：动手 Demo（可运行代码）

下面用 Python 模拟 Temperature 的作用。

Python

import random
import math

tokens = {
    "北京": 10,
    "上海": 8,
    "广州": 6,
    "火星": 2
}

def sample_token(logits, temperature):
    scaled = {
        k: v / temperature
        for k, v in logits.items()
    }

    exp_values = {
        k: math.exp(v)
        for k, v in scaled.items()
    }

    total = sum(exp_values.values())

    probs = {
        k: v / total
        for k, v in exp_values.items()
    }

    choices = list(probs.keys())
    weights = list(probs.values())

    return random.choices(
        choices,
        weights=weights,
        k=1
    )[0]

for temp in [0.5, 1.0, 2.0]:

    # 固定随机种子
    random.seed(42)

    print(f"\nTemperature={temp}")

    results = []

    for _ in range(20):
        results.append(
            sample_token(tokens, temp)
        )

    print(results)

Part 11：必背金句

[调试原则]：Temperature=0 是调试模式，不是正确模式。

[稳定原则]：输出稳定不代表答案正确。

[任务原则]：Temperature 应由业务目标决定。

[评测原则]：所有 Evaluation 必须固定 Temperature。

[影响原则]：Prompt、模型、RAG 的影响通常远大于 Temperature。
