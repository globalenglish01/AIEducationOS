## Section 2：薪资谈判的底层逻辑

---

### 为什么大多数候选人不谈薪资

在林雪准备薪资谈判时，她问过几个同期在找工作的朋友："你谈薪资吗？"

大多数人的回答是：

"公司给多少我就拿多少，我不想因为谈薪资显得太贪。"

"我害怕谈了之后公司不要我了。"

"我不知道我能要多少，不想说错了不好看。"

这三个回答背后有一个共同的前提假设：**谈薪资对我不利**。

林雪认为这个假设是错的。

---

### 薪资谈判的基本逻辑

**1. 发Offer不是最终价格**

公司发出的第一份Offer，通常是"HR认为在预算范围内、候选人大概率会接受"的数字——不是"我们愿意支付的最高数字"。

这不是说公司在欺骗你——只是Offer是一个起始点，不是终点。大多数公司都期待候选人会谈判，并且在预算里留有空间。

如果你不谈，公司省了钱，你少了本来可以拿到的薪资。没有人输，但你本来可以赢。

**2. 谈薪资不会导致Offer被收回（正常情况下）**

这是候选人最大的误解。

公司不会因为你"礼貌地问能不能更高"就撤掉Offer——这意味着他们白费了所有面试轮次的时间，以及HR的时间。

唯一可能让Offer被重新考虑的，是**态度**——如果你在谈判里显示出"不尊重"或者"漫天要价"，可能会改变公司对你的印象。但"我的预期是X，因为……，你们能不能调整"，不会有这个风险。

**3. 数据比感受更有说服力**

"我觉得我值更多"没有说服力。

"根据我对市场薪资的了解，类似岗位在这个城市的范围是X-Y，我的预期在这个范围的上段，因为……"，有说服力。

---

### 林雪的三个Offer谈判策略

**Offer A（大厂，薪资已是预期+20%）**

林雪的问题：薪资已经高于预期，有没有空间继续谈？

策略：**不谈基础薪资，谈其他方面**

她知道大厂的薪资结构通常是固定的，变动空间有限。但有其他可以谈的内容：

- **入职奖金（Sign-on Bonus）**：一次性发放，不影响基础薪资结构
- **股票期权的锁定时间（Vesting Schedule）**：能否加快前两年的锁定比例
- **入职时间的灵活性**：她还有另外两个Offer要处理，需要一点时间

实际话术：

"谢谢贵公司的Offer，薪资部分我很满意。我想问一下，入职奖金这方面是否有弹性？另外，关于入职时间，我还有另外两家在走流程，能否给我一周时间做综合决定？"

**Offer B（创业公司，薪资在预期线）**

林雪的问题：薪资在预期线，但想争取更多；同时期权怎么估值。

策略：**谈薪资 + 问清楚期权细节**

实际话术：

"我对公司的方向和团队非常感兴趣，这是我最想加入的方向。薪资方面，我的预期是X+10%，这是基于市场调研和我带来的经验密度。你们是否有空间调整？

另外，关于期权，我需要了解几个细节：
- 公司当前估值是多少？
- 期权池有多大（稀释风险）？
- Vesting Schedule是什么（通常是4年1年Cliff）？
- 如果公司未来融资，期权会怎么稀释？

这些信息会影响我对总体薪酬的判断。"

**Offer C（金融科技，薪资低于预期10%）**

林雪的问题：薪资低于预期，方向也不是最理想的。

策略：**直接提出需要薪资调整，否则很难接受**

实际话术：

"谢谢贵公司的Offer。方向上，我对保险系统AI化有深刻的理解，这个场景我很熟悉。但薪资方面，比我的预期低了X%，这个差距让我很难做决定——毕竟这次转型对我来说有一定的机会成本。

如果薪资能调整到Y，我愿意认真考虑。你们在这方面是否有空间？"

### 期权估值的简单框架

**林雪的实际计算（Offer B）**：

```
期权数量：公司给了林雪 N 份期权
当前公司估值：B轮后约X亿（HR告知的大概数字）
期权池占公司股权：约10%（B轮后常见范围）

理论上，N份期权当前纸面价值 = N/总股数 × X亿

但需要打折：
- 未上市流动性折扣：60-70%（卖不出去的股权没有价值）
- 稀释风险：后续融资轮会稀释，按C轮/D轮/上市来估计
- 行权价格：需要行权才能持有真正的股票，行权需要资金
```

林雪的结论：按保守估算，期权在5年视野里的期望价值相当于Y万元——不是零，但也不是大数字。在做决策时不应该把期权计入"确定的薪资"，但也不应该完全无视。

**建议**：初创公司的期权，当成"如果公司发展得好，会有额外收益"，而不是"这是我今天薪资的一部分"。

**💰 薪资 / 谈判相关**

| 术语 | 一句话解释 |
| --- | --- |
| **入职奖金（Sign-on Bonus）** | 入职时一次性发放的奖金，不包含在月薪里，常用来弥补其他损失。 |
| **Vesting Schedule** | 期权或股票分批解锁的时间表，不会入职就全部给你。 |
| **1年Cliff** | 入职满一年才能拿到第一批期权，不满一年离开则一分钱没有。 |
| **期权池** | 公司预留用于激励员工的股份总额。 |
| **稀释风险** | 后续融资会发行新股，你手上的股权占比会被摊薄。 |
| **行权价格** | 你把期权变成真正股票时需要支付的价格。 |
| **纸面价值** | 期权按当前估值算出来的理论价值，没上市前不是真钱。 |
| **流动性折扣** | 没上市的期权卖不掉，实际价值要打折扣。 |

**🎯 市场 / 估值相关**

| 术语 | 一句话解释 |
| --- | --- |
| **B轮后估值** | 公司完成B轮融资后整体值多少钱，用于估算期权价值。 |
| **期权锁定期** | 期权分批解锁的时间周期，通常4年全部解锁。 |
| **总股数** | 公司所有已发行股份的总数，用来计算期权的占比。 |
| **稀释影响** | 后续融资轮次会让你的股权比例变小，就是所谓的“被稀释”。 |
