### 一个实操建议

写简历时，做一个自我测试：

> 对着简历里的每一条，问自己：**"如果面试官继续追问这一条，我能说清楚三层吗？"**

- 第一层：这是什么？
- 第二层：为什么这样做而不是别的方式？
- 第三层：出了问题你怎么处理？

三层都答得出来的，写进简历。只能说第一层的，说明你理解不够深，暂时不要写。

## 📖 本章名词解释（新人必读）

> 第一次看到这些词？别慌，下面一句话搞定。

**🤖 AI 相关**

| 术语 | 一句话解释 |
| --- | --- |
| **RAG** | 让AI先查外部资料再回答，防止它瞎编的技术。 |
| **Agent** | 能自己决策、调用工具的AI“智能体”，像个数字员工。 |
| **LangGraph** | 用来构建多Agent协作流程的框架，像给Agent画流程图。 |
| **LightRAG** | 把知识图谱和向量检索结合的开源RAG框架。 |
| **LLM** | 大语言模型，像ChatGPT那样能理解和生成文字的AI大脑。 |
| **向量检索** | 把文字转成坐标，找意思最相近的内容，像找邻居。 |
| **知识图谱** | 用点和线表示实体和关系，像画人物关系图。 |
| **余弦相似度** | 用角度判断两个向量有多接近，像看指针方向是否一致。 |
| **Prompt工程** | 设计更精准的指令让AI按预期回答的技巧。 |
| **RAG评估** | 用指标衡量RAG系统检索和生成质量的过程。 |
