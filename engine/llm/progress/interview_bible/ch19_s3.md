## Section 3：谈判实录——林雪的三通电话

---

### 第一通电话：Offer A（大厂）

HR小刘打来电话，问林雪的决定。

**林雪**："谢谢贵公司的Offer，薪资部分我非常满意，在我的预期之上。我有两个问题想确认。

第一，入职奖金方面，公司是否有标准的入职奖金政策？如果有的话，我能了解大概范围吗？

第二，关于入职时间，我还有另外两家公司在走最后的流程，需要一周时间做综合决定。如果可能，我希望Offer的有效期能延长到下周五。"

**HR小刘**（停顿）："入职奖金这块，我们确实有，通常是X个月基础薪资。这个会在正式Offer里体现，我们应该没有漏掉……（翻查）……抱歉，确实漏掉了，我来更新一下Offer文件。入职时间方面，延到下周五可以，我会在系统里更新。"

**结果**：林雪发现入职奖金没有被包含在第一份Offer里——这是谈判让她多拿到了一笔钱，而且是HR遗漏了，不是林雪要求额外的。

---

### 第二通电话：Offer B（创业公司）

林雪主动打给HR向姐。

**林雪**："向姐，感谢你们给我Offer，这个方向是我最认同的。我想和你直接沟通几件事。

第一，薪资方面，我的预期是比目前Offer高X%。这个判断基于：我在保险行业有二十年经验，这直接对应你们的垂直行业方向，不需要行业学习成本；同时我有两个可以被追问的AI项目，不是刚转行的零经验。我认为这个调整是合理的。

第二，关于期权，我需要了解几个具体数字：现在的优先股估值是多少，期权池的规模，以及Vesting Schedule的具体安排。这些信息会影响我对总体薪酬的判断。"

**HR向姐**："薪资方面，我去和CEO确认一下，明天给你回复。期权的事我现在可以告诉你：Vesting是4年，前12个月Cliff，之后每月线性解锁。优先股估值……这个涉及公司内部信息，我需要请示一下是否可以透露。期权池目前占公司总股权的15%。"

**林雪**："好的，估值方面不方便的话，我理解。你能告诉我大概的范围就好——帮助我估算期权的参考价值。"

**结果**：次日，HR回复薪资可以调整5%（不是林雪要求的X%，但有调整）；优先股估值给了一个"不超过Y亿"的参考数字。林雪用这个数字做了大概估算。

---

### 第三通电话：Offer C（金融科技）

林雪打电话给HR李经理，语气直接。

**林雪**："李经理，感谢贵公司的Offer。我对这个方向有一些疑虑想和你沟通。

第一，薪资比我的预期低了大约X%。我理解每家公司有自己的薪资体系，但这个差距让我很难做决定——转型对我来说有真实的机会成本，如果薪资太低，我会怀疑这个选择是否划算。

第二，方向上，我注意到这个岗位很大一部分工作还是COBOL侧的整合，而不是AI工程的核心开发。这对我来说是一个需要权衡的地方——我在转型，希望能真正做AI工程，而不是'带一点AI的传统IT'。

我想直接问：薪资方面能否调整到Y？以及，这个岗位6个月后有没有可能往更纯粹的AI工程方向发展？"

**HR李经理**（停顿）："薪资方面，我需要和部门负责人确认，但我个人认为调整空间有限——我们岗位的薪资体系是统一的。方向上，你说的是对的，这个岗位确实是混合型的，纯AI工程的占比大约是三四成。"

**结果**：李经理次日回复薪资只能调整2%（林雪要求的差距无法弥合）。林雪决定不接这个Offer。

---

### 林雪的最终选择

三通电话之后，林雪手里的实际情况：

- Offer A：薪资（含入职奖金）高于预期25%，稳定，但方向偏平台
- Offer B：薪资调整到预期+5%，方向最契合，但公司风险存在
- Offer C：方向不理想，薪资无法调整到位，放弃

选择Offer A还是Offer B？

林雪回到那张A4纸，看了看她写的三条不可妥协的条件：

1. 薪资不低于现在：两者都过
2. 工作方向是AI工程：两者都过（A偏平台，B偏应用，都算AI工程）
3. 可见的未来有成长机会：A更确定，B取决于公司发展

然后她问了自己一个不在条件列表里的问题：

**"如果两家公司都发展顺利，三年后，哪家给我的成长更有价值？"**

答案让她做了决定。

## 📖 本章名词解释（新人必读）

> 第一次看到这些词？别慌，下面一句话搞定。

**🤖 AI 相关**

| 术语 | 一句话解释 |
| --- | --- |
| **AI工程** | 构建和部署AI系统（如Agent、RAG）的软件开发实践。 |
| **AI平台** | 为多个AI应用提供底层模型调用、监控等基础服务的系统。 |
| **AI应用** | 直接面向业务场景的AI功能，如智能客服、自动化测试。 |

**💼 金融 / 保险行业相关**

| 术语 | 一句话解释 |
| --- | --- |
| **COBOL侧整合** | 把COBOL老系统和新的AI模块对接起来的工作。 |
| **传统IT** | 传统的企业信息系统开发与维护，区别于AI/云原生等新技术方向。 |

**💰 薪资 / Offer相关**

| 术语 | 一句话解释 |
| --- | --- |
| **入职奖金** | 签约时一次性发放的额外奖金，不算在月薪里。 |
| **Offer有效期** | 公司给出的决定期限，逾期Offer自动失效。 |
| **优先股估值** | 创业公司融资时的每股价格，用来估算期权的价值。 |
| **期权池** | 公司预留用于激励员工的股份总额，占股比例决定你的稀释程度。 |
| **Vesting Schedule** | 期权分批解锁的时间表，不会入职就一次性给你全部。 |
| **1年Cliff** | 必须做满一年才能拿到第一批期权，提前离开则全部归零。 |
| **线性解锁** | 过了Cliff期后，剩余期权按月均匀分批释放给你。 |
| **薪资体系** | 公司内部的薪酬等级和带宽制度，每档有固定的上下限。 |

**🎯 职场 / 决策相关**

| 术语 | 一句话解释 |
| --- | --- |
| **垂直行业方向** | 聚焦特定行业（如保险、医疗）的AI应用，不是通用型AI。 |
| **行业学习成本** | 新入职员工熟悉业务、理解行业术语所需的时间。 |
| **混合型岗位** | 同时做传统IT和AI两方面工作的职位。 |
| **机会成本** | 选择转行AI而放弃原有职业路径所失去的确定收入。 |
